ПРОМТ ДЛЯ СОЗДАНИЯ САЙТА-ВИЗИТКИ РЕПЕТИТОРА

Создай современный, профессиональный сайт-визитку для репетитора с использованием HTML, CSS и JavaScript. Дизайн должен быть основан на успешных образовательных платформах типа Coursera, Khan Academy и лучших личных сайтах преподавателей.

ТЕХНИЧЕСКОЕ ЗАДАНИЕ:

1. СТРУКТУРА САЙТА:
- Главная секция с героем (имя, фото, основная специализация)
- Секция "Обо мне" (образование, опыт, достижения)
- Секция "Услуги" (предметы, форматы занятий, цены)
- Секция "Результаты" (отзывы учеников, статистика успеха)
- Секция "Контакты" (форма записи, способы связи)

2. ДИЗАЙН-ТРЕБОВАНИЯ:
- Цветовая схема: глубокий синий (#1e3a8a) как основной, светло-голубой (#dbeafe) как акцент, белый фон
- Типографика: использовать Google Fonts (Montserrat для заголовков, Open Sans для текста)
- Минималистичный, но теплый дизайн
- Адаптивная верстка для всех устройств
- Плавные анимации и переходы

3. КЛЮЧЕВЫЕ ЭЛЕМЕНТЫ:
- Профессиональное фото репетитора в header
- Иконки для предметов и услуг (математика, физика, химия и т.д.)
- Карточки с отзывами в слайдере
- Калькулятор стоимости занятий
- Календарь для записи на пробный урок
- Интеграция с WhatsApp и Telegram

4. КОНТЕНТ:
- Заголовок: "Частный репетитор [Предмет] - Качественная подготовка к экзаменам"
- Подзаголовок о результатах (например: "95% учеников улучшают оценки на 2+ балла")
- Блок с достижениями (стаж, количество учеников, средний балл ЕГЭ)
- Описание методики преподавания
- Пакеты услуг с разными тарифами

5. ИНТЕРАКТИВНОСТЬ:
- Плавный скролл между секциями
- Модальные окна для подробной информации
- Форма обратной связи с валидацией
- Счетчики достижений с анимацией
- Hover-эффекты на карточках и кнопках

6. SEO И ПРОИЗВОДИТЕЛЬНОСТЬ:
- Семантическая разметка HTML5
- Мета-теги для поисковых систем
- Оптимизированные изображения
- Быстрая загрузка страницы

СТИЛЬ КОДИРОВАНИЯ:
- Чистый, комментированный код
- Использование современных CSS (Flexbox, Grid)
- JavaScript ES6+
- Адаптивные единицы измерения (rem, em, vw, vh)

Создай полнофункциональный сайт с реалистичным контентом, который можно сразу использовать для продвижения услуг репетитора.
